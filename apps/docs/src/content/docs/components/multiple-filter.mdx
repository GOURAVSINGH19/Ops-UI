## Overview

The MultipleFilter (internal name `Upgradefilter`) is a specialized dropdown component designed for handling complex filtering requirements. It supports grouping options, multi-selection with visual feedback (checkmarks), and displays a selection count badge on the main trigger.

## Features

<Callout type="info">
  Key features of the MultipleFilter component:
  - **Grouped Options**: Organizes filter options into distinct categories.
  - **Multi-select**: Allows selecting multiple values across different groups.
  - **Selection Counter**: Automatically displays a badge with the total number of items selected.
  - **Visual Feedback**: Active options are highlighted with a background color and checkmark.
  - **Click Outside to Close**: Automatically closes the dropdown when the user clicks elsewhere.
  - **Glassmorphism Trigger**: Sophisticated trigger design with backdrop blur.
</Callout>

## Best Practices

1. **Logical Grouping**: Group related filters together (e.g., "Status", "Priority", "Project").
2. **Descriptive Labels**: Use clear labels for both the trigger and the individual options.
3. **Handle State**: Ensure your filtration logic can handle an empty selection as a "select all" or "no filter" state.
4. **Z-Index**: Be mindful of the surrounding UI to ensure the dropdown doesn't get clipped by other containers.
