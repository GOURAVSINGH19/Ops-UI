The SpiderView component is ideal for comparing multiple quantitative variables. It is commonly used for performance metrics, system health comparisons, or feature set analysis.

## Features

- **Multi-Axis Comparison**: Visualize relationships between multiple dimensions effortlessly.
- **Interactive Data Points**: Hover to see precise values and category labels.
- **Customizable Scales**: Adjust axis ranges and grid increments.
- **Responsive Sizing**: Automatically scales to fit its parent container.
