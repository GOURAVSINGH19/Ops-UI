# Getting Started

Welcome to the component library! This guide will help you get started with installing and using our components.

<Callout type="info" title="Quick Start">
  If you're new to this library, we recommend starting with the
  [Introduction](/docs/introduction) page to understand our approach.
</Callout>

## Prerequisites

Before you begin, make sure you have:

- Node.js 18+ installed
- Basic knowledge of React and TypeScript

## Installation

<Callout type="warning" title="Important">
  Make sure your project is set up with Tailwind CSS before installing
  components. Our components rely on Tailwind for styling.
</Callout>

## Next Steps

Once installed, you can:

1. Installation :
   <InstallationCli value="npm i @demouser22/opscli" />
2. Add component
   <InstallationCli value="opscli add <component_name>" />

<Callout type="success" title="Ready to go!">
  You're all set! Start exploring components and building amazing interfaces.
</Callout>
